## Installation
```bash
# Clone the repository
git clone https://github.com/Almazatun/qrcode-iod.git
# Enter into the directory
cd qrcode-iod/
# Install the dependencies
go mod download | go mod tidy | make install
```

### Build app
```bash
$ make build
```
### Run app
```bash
$ make run
```
