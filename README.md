## Installation
```bash
# Clone the repository
git clone https://github.com/Almazatun/qrcode-iod.git
# Enter into the directory
cd qrcode-iod/
# Install the dependencies
make install
```

### Build web app
```bash
$ make build_web
```
### Run web app 
```bash
$ make run_web
```
